isis :: 。牽羊悔亡，聞言不信
Monday, February 21st 2005
。牽羊悔亡，聞言不信 | 10:32 AM

因為站起來是那麼偶然那麼難，每次感覺要跌的時候，便待在低的地方，想我是多麼好奇多麼懷疑多麼善於絕決也多麼容易崩潰。但常失而復得。

「年輕時我相信 / 各有各的煉獄，活到這把年紀 / 知道它們最終也無能彼此 / 救贖。枉我不放心一再追究 / 一再深入 」

「深入後所失去的表面---- / 表面藏著一道拉鍊，清楚聽見 / 拉鍊滑下的聲音 / 打開一片肉遍佈透明的 / 觸鬚拔下一根 / 吹一口氣 / 滿山遍野一大群羊顯現----」

「我的狗用牠濕冷的鼻子碰我， / 喔，來人要卜，三個銅錢六次 / 搖落，夬卦九四爻： / 臀無膚 / 其行次且 / 牽羊悔亡 / 聞言不信」


（不信則無。自知之明。）
（感謝從小撫養我並且流浪著把命交給陌生人陌生鬼陌生神的家族培養了我極大的厭惡。）
（我知道，我不說。那些預言被我扔了進垃圾桶。）
（我最最親愛的Debby去了哪裡我太想念妳。妳還會在我哭泣時擁著我嗎。我看不見。那些卦象像妳的腳印，不復存在。）
（徘徊的夜流連的車底工地電話亭床下衣櫥就過去了吧。再低不過如此。）
〈牽羊悔亡〉 夏宇
-----給一隻叫混（1982-1997）的狗

為什麼剛好
住在這棟樓裡 37號綠色的門
門的密碼是大革命那一年
我不知道為什麼
剛好我浮了起來在5樓朝北
的房間裡被鄰人瞧見
他也並不吃驚
他看了我幾秒鐘離開窗口把電視機關掉

其他鄰人評論道，關於我騰空浮起一事
她已經那麼老了
她可以為所欲為
更何況，其中一個說
她已經試了一輩子

只有我的狗表示錯愕與不滿
牠明白像這樣牠很難跟我一起散步
我慢慢降回地面的時候
我們都睏極了
睡一大覺後收拾行李
帶牠出門旅行

抽著用葉子捲好的土煙
我滿臉皺紋一頭白髮
牠坐在我的旁邊我們等人來卜
我把三個銅板擲開牠大叫數聲
都各有旨意
沿地一排64個信封都是我年輕時為卦寫的詩
那些卦象像牠的腳印

這是偏遠荒遼的谷地
看那些草略茫昧月湧星垂
我狂喜狂悲進退皆險
真真不如市集裡村人
以物易物自生自滅 肉與字
年輕時我相信
各有各的煉獄，活到這把年紀
知道它們最終也無能彼此
救贖。枉我不放心一再追究
一再深入

深入後所失去的表面----
表面藏著一道拉鍊，清楚聽見
拉鍊滑下的聲音
打開一片肉遍佈透明的
觸鬚拔下一根
吹一口氣
滿山遍野一大群羊顯現----

我的狗用牠濕冷的鼻子碰我，
喔，來人要卜，三個銅錢六次
搖落，夬卦九四爻：
臀無膚
其行次且
牽羊悔亡
聞言不信
